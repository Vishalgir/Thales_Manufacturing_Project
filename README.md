# Manufacturing Process Health and Operational Efficiency Analysis

This project analyzes manufacturing process health and operational efficiency in a 6G-enabled smart factory environment.

## Project Objectives
- Analyze machine health using temperature, vibration, and power consumption.
- Identify production inefficiencies.
- Analyze defect rate and error rate.
- Compare efficiency status across machines and operation modes.
- Build an interactive Streamlit dashboard.

## Dashboard Features
- Factory Health Overview
- Machine Health Dashboard
- Production & Quality Panel
- Efficiency Diagnostics
- Machine filter
- Operation mode filter
- Date range filter

## Technologies Used
- Python
- Pandas
- Plotly
- Streamlit
